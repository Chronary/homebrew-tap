# Chronary Homebrew Tap

Homebrew formula for the [Chronary](https://chronary.ai) CLI.

## Install

```bash
brew install chronary/tap/chronary
```

Or in two steps:

```bash
brew tap chronary/tap
brew install chronary
```

## Update

```bash
brew update
brew upgrade chronary
```

## What's `chronary`?

A command-line tool for the Chronary calendar-as-a-service API. Source and issues: https://github.com/Chronary/chronary-cli

## How this tap stays in sync

This formula is auto-generated by the Chronary CLI release pipeline on every CLI release. Manual edits will be overwritten on the next release.
