# Chronary Homebrew Tap

Official Homebrew tap for the [Chronary](https://chronary.ai) CLI.

## Install

\`\`\`bash
brew tap Chronary/tap
brew install chronary
\`\`\`

## Source

Formulae in this repo are auto-published by the release workflow of
[Chronary/chronary-cli](https://github.com/Chronary/chronary-cli). Do not
edit \`Formula/chronary.rb\` by hand — changes will be overwritten on the
next CLI release.
